The Strategy Listing screen allows you to access and manage the strategies created in your account. From the listing screen, you can request to create a new strategy, edit, view details, preview, search, archive and delete a strategy.

## Getting to Strategy Listing

**Navigation:**

  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/strategylisting1b.png" alt="Strategy Listing" title="Strategy Listing" />
</div>

- From the Vue Home Page -> Click on Digital Experience Hub to visit the Digital Experience Home Screen.
- Click on Assets on the navigation bar and select Strategies from the drop-down menu to access the Strategy Listing screen.
- Alternatively, hover over the Digital Experience Hub on the navigation bar, click on Assets, and select Strategy.

## Search Strategy

1. You can search for the created strategy using the strategy name via the search bar or use one of the suggested/recently searched keywords.
2. You can find the search results populated in the table.

## Sort Strategy

1. Hover over the column header to find the sort icon next to each column.
2. Click the Sort icon to sort the column alphabetically either in ascending or descending order.
3. If the sort is applied to multiple columns, the column for which the sort was applied last will take precedence.

## Filter Strategy

1. Hover over the column header to find the filter icon next to each column.
2. Click the filter icon to open up a dropdown from which you can multi-select the values to be filtered.
3. You can also search for a filter value within the dropdown.
4. The table will be populated with the filtered results.

## Delete Strategy

1. From the strategy listing table, next to each strategy name, click on the ‘Delete’ icon under actions.
2. Deleting a strategy will permanently remove it from Vue.
3. If the strategy is linked to other items, click the Request button to fill out the form provided to submit a request for deletion. We will get back to you in 5-7 days.

**Note:** The user is not allowed to delete a ‘Live’ strategy.

## Viewing a Strategy Config

1. From the strategy listing table, next to each strategy name, click on the ‘Info’ icon under actions.
2. You can also access all the entities linked using the different tabs mentioned below:
   1. **Details** (Lists all the general info of the strategy):
      - Name
      - Model
      - Model Parameter
      - Selected Catalog
      - 1:1 Personalization
   2. **Modules** (Lists all the module details linked to the strategy):
      - Module Name
      - Status
      - Integration Mode
      - Modified Date
   3. **Experiences** (Lists all the experience details linked to the strategy):
      - Experience Name
      - Status
      - Modified Date
