<p>Creating a workflow in Vue is an efficient solution to automate processes and tasks that can be set to run as per your requirement.</p>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-103334.png" alt="Workflow Creation" title="Workflow Creation" />
</div>

<h2>Steps to Create a Workflow:</h2>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-103800.png" alt="Initiate Workflow Creation" title="Initiate Workflow Creation" />
</div>

<h3>Initiate Workflow Creation:</h3>
<ol>
  <li>Navigate to the Workflows Listing Screen.</li>
  <li>Click the ‘+ New Workflow’ to create a Fresh Workflow Canvas.</li>
</ol>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-065042.png" alt="New Workflow" title="New Workflow" />
</div>

<h3>Renaming the Workflow:</h3>
<p>Click on the Edit button to change the name of your Workflow as needed.</p>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-065251.png" alt="Rename Workflow" title="Rename Workflow" />
</div>

<h3>Building a Workflow:</h3>
<h4>Add Nodes:</h4>
<p>Nodes can be added to the workflow in two ways:</p>
<ul>
  <li>Add a node by hovering over the Nodes section, searching for the needed Node, and finally dragging & dropping it into the Workflow Canvas.</li>
  <li>Right-click on the Workflow Canvas and search for the required node to add it. This also shows the history of recent nodes used.</li>
</ul>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-071522.png" alt="Add Nodes" title="Add Nodes" />
</div>

<h4>Link Nodes:</h4>
<p>After adding the required nodes, they can be linked by dragging the end of one node and connecting it to the start of another node.</p>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-074222.png" alt="Link Nodes" title="Link Nodes" />
</div>

<h3>Save and Deploy:</h3>
<p>Click on ‘Save’ once the Workflow is built. The Workflow is now ready to be Deployed!</p>
