<p>Nodes are the building blocks of any workflow. Based on the use-case of the workflow that is to be built, different types of nodes can be added. A quick primer on how to use nodes is covered below:</p>

<h2>Steps to Add Nodes in the Canvas:</h2>

<h3>To find the required Node:</h3>
<ul>
  <li><strong>Hover over the Nodes section</strong> to the left of the Canvas, drag & drop the node needed to the Canvas.</li>
</ul>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-094008.png" alt="Drag and Drop Node" title="Drag and Drop Node" />
</div>

<ul>
  <li><strong>Right-Click on the Canvas space</strong> to search for the Node required, this also shows the latest history of nodes used by you.</li>
</ul>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-094149.png" alt="Right-Click Search" title="Right-Click Search" />
</div>

<h3>Add the Node to the Canvas:</h3>
<ul>
  <li>Add the Node to the Canvas, to be used in the Workflow.</li>
</ul>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-082614.png" alt="Add Node" title="Add Node" />
</div>

<h2>Steps to Configure Nodes:</h2>
<ul>
  <li><strong>Click on the added Node</strong> to open the Forms to Customize it.</li>
</ul>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-082738.png" alt="Customize Node" title="Customize Node" />
</div>

<p>The Node can be customized as needed. After which, click 'Add' in order to save the customizations & add the updated node in the Workflow Canvas.</p>

<p>Different types of Nodes available in the Workflow Builder consist of different Fields in the Customization Form which can be tailored as per your use-case.</p>
