<p>In Audience Manager, view the list of “All Audiences” created in a table. From here you can either ‘Upload’ a Custom Audience or view Audience configuration & overlap details.</p>

<h2>Getting to Audience Listing</h2>

<h3>Navigation:</h3>

<ul>
  <li>From the Vue Home Page -> Click on Audience Hub to visit the Audience Home Screen.</li>
  <li>Click on Audience Manager on the navigation bar to access the Audience listing table.</li>
  <li>Alternatively, Hover over the Audience Hub on the navigation bar and click on Audience Manager.</li>
</ul>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/audience1b.png" alt="Audience Listing" title="Audience Listing" />
</div>

<h2>Search Audience</h2>

<ol>
  <li>You can search for the created Audiences using the Audience name via the search bar or use one of the recently searched keywords.</li>
  <li>You will find the search results populated in the table.</li>
</ol>

<h2>Sort Columns</h2>

<ol>
  <li>Hover over the column header to find the sort icon <button style={{border: "none", backgroundColor: "transparent"}}><img src="https://d1r1e7xjkfj7nz.cloudfront.net/Sort.png" /></button> next to each column</li>
  <li>Click the Sort icon <button style={{border: "none", backgroundColor: "transparent"}}><img src="https://d1r1e7xjkfj7nz.cloudfront.net/Sort.png" /></button> to sort the column alphabetically either in ascending or descending order.</li>
  <li>If the sort is applied to multiple columns, the column for which sort was applied last will take precedence.</li>
</ol>

<h2>Filter Columns</h2>

<ol>
  <li>Hover over the column header to find the filter icon <button style={{border: "none", backgroundColor: "transparent"}}><img src="https://d1r1e7xjkfj7nz.cloudfront.net/Filters.png" /></button> next to each column</li>
  <li>Click the filter icon <button style={{border: "none", backgroundColor: "transparent"}}><img src="https://d1r1e7xjkfj7nz.cloudfront.net/Filters.png" /></button> to open up a dropdown from which you can multi-select the values to be filtered.</li>
  <li>You can also search for a filter value within the dropdown.</li>
  <li>The table will be populated with the filtered results.</li>
</ol>

<h2>Upload a Custom Audience</h2>

<ol>
  <li>Navigate to Audience Hub -> Audience Manager</li>
  <li>Click on ‘Upload Audience’ <i>Note: Download the .csv template and populate the template with the required data (Optional)</i></li>
  <li>Either drag & drop a CSV file with audience data or browse and upload the file from a local location.</li>
  <li>Enter a name and description for your Audience.</li>
  <li>Click on ’Upload’.</li>
  <li>You can view the uploaded audience and associated data in the Audience listing table.</li>
</ol>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/audience2b.png" alt="Upload Audience" title="Upload Audience" />
</div>

<h2>Checking Performance Data</h2>

<p>In the Audience listing table, next to each audience name, by default, you will find data about the:</p>

<table>
  <thead>
    <tr>
      <th>Metrics</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Conversion Rate</td>
      <td>Number of Orders placed per Visit</td>
    </tr>
    <tr>
      <td>Revenue Per Visitor</td>
      <td>Total Revenue generated per Visit</td>
    </tr>
    <tr>
      <td>AOV</td>
      <td>Average Revenue generated per Order</td>
    </tr>
    <tr>
      <td>Unique Visitors</td>
      <td>Number of unique visitors</td>
    </tr>
    <tr>
      <td>Last Run Date</td>
      <td>The date at which the audience has been last updated</td>
    </tr>
  </tbody>
</table>

<h2>Checking Overlap Information</h2>

<ol>
  <li>In the Audience listing table, click on the ‘Overlap’ icon next to the audience name</li>
  <li>You can view overlapping details for the selected Audience against all other available audiences (illustrated below).</li>
</ol>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/audience3b.png" alt="Audience Overlap" title="Audience Overlap" />
</div>

<h2>Exporting an Audience</h2>

<p>In the Audience listing table, click on the export icon beside the audience to be exported to export the audience as a CSV.</p>
