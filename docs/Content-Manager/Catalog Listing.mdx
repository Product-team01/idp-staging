<p>Catalogs created by you can be visualized in the Listing Table in the Explore Screen. The listing enables you to manage, search, sort, filter, and delete catalogs.</p>

<h2>Navigation</h2>
<ul>
  <li>From the Vue Home Page -> Click on Content Hub to visit the Content Home Screen.</li>
  <li>Click on Explore on the navigation bar to access the Catalog listing table.</li>
  <li>Alternatively, Hover over the Content Hub on the navigation bar and click on Explore.</li>
</ul>

<h2>Catalog Listing</h2>
<p>View the list of “All Catalogs” created in a table. From here you can either create a new Catalog or manage an existing Catalog and perform actions like edit, archive, delete.</p>

<h2>Search Catalogs</h2>
<ol>
  <li>You can search for the created Catalogs using the Catalog name via search bar or use one of the suggested/recently searched keywords.</li>
  <li>You can find the search results populated in the table.</li>
</ol>

<h2>Sort Columns</h2>
<ol>
  <li>Hover over the column header to find the sort icon next to each column.</li>
  <li>Click the Sort icon to sort the column alphabetically either in ascending or descending order.</li>
  <li>If the sort is applied to multiple columns, the column for which sort was applied last will take precedence.</li>
</ol>

<h2>Filter Columns</h2>
<ol>
  <li>Hover over the column header to find the filter icon next to each column.</li>
  <li>Click the filter icon to open up a dropdown from which you can multi-select the values to be filtered.</li>
  <li>You can also search for a filter value within the dropdown.</li>
  <li>The table will be populated with the filtered results.</li>
</ol>

<h2>Delete Catalog</h2>
<ol>
  <li>From the Catalog listing table, next to each catalog name, click on the Delete icon under actions.</li>
  <li>Clicking Delete will prompt the user with an overlay modal that lists all the entities (strategies and projects) the catalog is linked with.</li>
  <ul>
    <li>Here, you can access any entity config screen and make necessary changes before deleting the Catalog.</li>
  </ul>
  <li>Deleting the catalog will unlink it from all the linked features and permanently delete it & the content from our system.</li>
</ol>

<h2>Archive a Catalog</h2>
<ol>
  <li>Catalogs can be archived by clicking the Archive icon under actions.</li>
  <li>Clicking Archive will prompt the user with a pop-up displaying all the services (strategies and projects) the catalog is linked with.</li>
  <li>Archiving the catalog will unlink it from all the mentioned services.</li>
</ol>

<h2>Suggested Catalog</h2>
<ol>
  <li>From the suggested catalog section, you can select a catalog with a preselected configuration.</li>
  <li>There are 4 pre-set catalog types currently supported in Vue:</li>
  <ul>
    <li>Product Catalog</li>
    <li>Banner Catalog</li>
    <li>Offer Catalog</li>
    <li>Documents</li>
  </ul>
</ol>
