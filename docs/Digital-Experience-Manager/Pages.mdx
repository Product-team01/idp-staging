<p>The different pages on a website such as the home page, product listing page, product detail page, cart page, etc. can be configured to be used in experiences.</p>

<h2>Getting to Pages</h2>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/pagelisting1b.png" alt="Pages" title="Pages" />
</div>

<p><strong>Navigation</strong></p>

<ol>
  <li>To access Pages navigate to the Vue menu bar, click on this icon</li>
  <li>Select DXM</li>
  <li>Select Pages</li>
</ol>

<h2>Pages Listing Screen</h2>

<p>The Page Listing screen provides you with the list of Pages created in your account. From the listing screen you can request creation of a new Page, view details about each Page configuration, Preview, and Delete a Page.</p>

<ul>
  <li>Page Listing</li>
  <li>Search Pages</li>
  <li>Sort Columns</li>
  <li>Filter Columns</li>
  <li>Delete Page</li>
</ul>

<h3>Search Pages</h3>

<ol>
  <li>You can search for the created Pages using the Page name via search bar or use one of the suggested/recently searched keywords.</li>
  <li>You can find the search results populated in the table.</li>
</ol>

<h3>Sort Columns</h3>

<ol>
  <li>Hover over the column header to find the sort icon next to each column</li>
  <li>Click the Sort icon to sort the column alphabetically either in ascending or descending order.</li>
  <li>If the sort is applied to multiple columns, the column for which sort was applied last will take precedence.</li>
</ol>

<h3>Filter Columns</h3>

<ol>
  <li>Hover over the column header to find the filter icon next to each column</li>
  <li>Click the filter icon to open up a dropdown from which you can multi-select the values to be filtered.</li>
  <li>You can also search for a filter value within the dropdown.</li>
  <li>The table will be populated with the filtered results.</li>
</ol>

<h3>Delete Pages</h3>

<ol>
  <li>From the page listing table, next to each page name, click on the Delete icon under actions.</li>
  <li>Clicking Delete will prompt the user with an overlay modal which lists all the entities (experiences) the page is linked with.
    <ul>
      <li>Here, you can access any entity config screen, make necessary changes before deleting the page.</li>
    </ul>
  </li>
  <li>Deleting the page will unlink it from all the linked experiences and permanently delete it & the content from our system</li>
</ol>

<h3>Viewing Page Details</h3>

<ol>
  <li>From the Page listing table, next to each Page name, click on the 'Info' icon under actions.</li>
  <li>Page configuration details will open in an overlay modal.</li>
  <li>From here, you can also access all the entities linked (Experience) to this Page and navigate to them.</li>
  <li>You can also view a preview of the entire page with all the configured placements.</li>
  <li><strong>Note:</strong> Page Preview is only supported for Desktop & Mobile device types.</li>
</ol>

<h3>Request Creation of a New Page</h3>

<ol>
  <li>Navigate to Digital Experience Manager (DXM) via Vue Menu Bar and click on 'Assets' > Page</li>
  <li>Click on 'Request New'</li>
  <li>In the form provided please fill out the necessary details. And we will get back to you with your Page within 7 to 14 business days.</li>
</ol>
