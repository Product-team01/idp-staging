  <h3>Admin Users</h3>
  <p>Admin users can access the User Roles tab to create roles, assign permissions, and manage account access across their team:</p>

  </div>
  <ul>
    <li>Click on ‘Manage User Roles’ from the side navigation of your Account Settings.</li>

      <img src="https://d1r1e7xjkfj7nz.cloudfront.net/count4.png" alt="New Role" title="New Role" />
    </div>
    <li>Here, you will see existing roles or have the option to create new ones.</li>
    <li>To create a new role, click the ‘+ New Role’ button. Provide a Name and/or Description for your new role, then assign access permissions per entity as required. Click “Save” when you’re ready to return to the listing.</li>

      <img src="https://d1r1e7xjkfj7nz.cloudfront.net/count5.png" alt="Role Actions" title="Role Actions" />

  <h3>Users & User Groups</h3>
  <p>Admin users can manage Users and User Groups from the respective tabs within their Account Settings:</p>
  <div className="image-container">

  </div>
  <ul>
    <li>To navigate, click on the ‘Manage Users’ or ‘Manage User Groups’ tab using the side navigation of your Account Settings.</li>
    <li>To create a new User, click the ‘+ New User’ button on the Manage Users tab. Enter relevant information such as Name & Credentials, assign access roles & permissions, and click “Create” when you’re ready.</li>
    <li>To create a new User Group, click the ‘+ New User Group’ button on the Manage User Groups tab. Enter a Group Name, select the Users to include in this group, assign access roles and permissions, and click “Create” when you’re ready.</li>
    <div className="image-container">
      <img src="https://d1r1e7xjkfj7nz.cloudfront.net/count7.png" alt="User Configurations" title="User Configurations" />

  <h2>Assignment</h2>
  <p>Admin users can assign entities to other users and/or user groups as follows:</p>
  <ul>
    <li>Navigate to the Entity Listing, where you will see an ‘Assign User’ column.</li>
    <li>To assign users at a row level, use the dropdown to select User(s) and/or User Groups.</li>
    <li>To bulk assign entities, multi-select the required entities and use the ‘Assign Users’ button above the listing to select User(s) and/or User Groups.</li>
    <li>Non-admin users will only be able to view the entities assigned to them on the relevant listings.</li>
  </ul>
