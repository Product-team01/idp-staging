<h2>Sort Columns</h2>
<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-105402.png" alt="Sort Columns" title="Sort Columns" />
</div>
<p>Hover on the column header to find the sort option to the right of a column header.</p>
<p>Click the Sort icon to sort the column alphabetically in either ascending or descending order.</p>
<p>If the sort is applied to multiple columns, the column for which the sort was applied last will take precedence.</p>

<h2>Filter Columns</h2>
<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-105516.png" alt="Filter Columns" title="Filter Columns" />
</div>
<p>Hover on the column header to find the filter option to the left of a column header.</p>
<p>Click the filter icon to open up a dropdown from which you can single select or multi-select the values to be filtered.</p>
<p>You can also search for a filter value within the dropdown.</p>

<h2>Delete Workflow</h2>
<p>From the Workflow listing table, next to each Workflow name, click on the Delete icon under actions.</p>
<p>Clicking Delete will prompt the user with an overlay modal to confirm this action.</p>
<p>Deleting the Workflow will permanently remove it and the content from the system.</p>
