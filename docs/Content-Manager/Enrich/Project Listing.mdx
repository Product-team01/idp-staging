<p>View the list of “All Projects” created in a table. From here you can either create a new Project or manage an existing Project and perform actions like edit, archive, and delete.</p>

<h2>Getting to Project Listing</h2>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/projectlisting1b.png" alt="Project Listing" title="Project Listing" />
</div>

<h2>Navigation:</h2>
<ul>
  <li>From the Vue Home Page -&gt; Click on Content Hub to visit the Content Home Screen.</li>
  <li>Click on Enrich on the navigation bar to access the Project listing table.</li>
  <li>Alternatively, Hover over the Content Hub on the navigation bar and click on Enrich.</li>
</ul>

<h2>Search Project</h2>
<ol>
  <li>You can search for the created Projects using the Project name via the search bar or use one of the suggested/recently searched keywords.</li>
  <li>You can find the search results populated in the listing table.</li>
</ol>

<h2>Sort Columns</h2>
<ol>
  <li>Hover over the column header to find the sort icon next to each column.</li>
  <li>Click the Sort icon to sort the column alphabetically either in ascending or descending order.</li>
  <li>If the sort is applied to multiple columns, the column for which the sort was applied last will take precedence.</li>
</ol>

<h2>Filter Columns</h2>
<ol>
  <li>Hover over the column header to find the filter icon next to each column.</li>
  <li>Click the filter icon to open up a dropdown from which you can multi-select the values to be filtered.</li>
  <li>You can also search for a filter value within the dropdown.</li>
  <li>The table will be populated with the filtered results.</li>
</ol>

<h2>Delete Project</h2>
<ol>
  <li>From the Project listing table, next to each Project name, click on the Delete icon under actions.</li>
  <li>Clicking Delete will prompt the user with an overlay modal which lists all the entities (strategies and catalogs) the Project is linked with.</li>
  <li>Here, you can access any entity config screen, make necessary changes before deleting the Project.</li>
  <li>Deleting the Project will unlink it from all the linked features and permanently delete it & the content from our system.</li>
</ol>
