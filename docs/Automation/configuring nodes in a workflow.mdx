Nodes are the building blocks of any workflow. Based on the use-case of the workflow that is to be built, different types of nodes can be added. A quick primer on how to use nodes is covered below:

## Steps to Add Nodes in the Canvas:

### To find the required Node:
- **Hover over the Nodes section** to the left of the Canvas, drag & drop the node needed to the Canvas.

![Drag and Drop Node](https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-094008.png)

- **Right-Click on the Canvas space** to search for the Node required, this also shows the latest history of nodes used by you.

![Right-Click Search](https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-094149.png)

### Add the Node to the Canvas:
- Add the Node to the Canvas, to be used in the Workflow.

![Add Node](https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-082614.png)

## Steps to Configure Nodes:
- **Click on the added Node** to open the Forms to Customize it.

![Customize Node](https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-082738.png)

- The Node can be customized as needed. After which, click 'Add' in order to save the customizations & add the updated node in the Workflow Canvas.

Different types of Nodes available in the Workflow Builder consist of different Fields in the Customization Form which can be tailored as per your use-case.
