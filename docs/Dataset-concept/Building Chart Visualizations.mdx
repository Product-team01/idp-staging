Creating chart visualizations involves transforming raw data into visual formats that highlight key insights and trends. Below Documentation will walk you through building the different types of charts available in the Metrics Visualizer. 

If you need help determining the right chart for your data, refer to this documentation for guidance 


<div class="partner-details">
  

  <h2>Bar / Area / Line Chart</h2>
  <details>
    <summary>Configuring Your Visual</summary>
    <p>Title: Set a concise title that describes your visual's purpose.</p>
    <p>Dataset: Choose the dataset on which your visual will be based.</p>
    <p>If you've uploaded a new dataset, click the refresh button to ensure it's up-to-date and available for visualization.</p>
    <p>Visual Type: Select the type of visual you prefer: Bar, Line, or Area Chart.</p>
    <p>Filters: Apply filters to narrow down the data for your insights. Combine multiple filters using 'and' or 'or' operators.</p>
    <p>Operation: Choose a conditional operator (e.g., ==, >=) for your filter.</p>
    <p>Field: Select the column to which you want to apply the filter.</p>
    <p>Value: Enter the filter value to refine your data selection.</p>
  </details>

  <details>
    <summary>Setting up the X-Axis and Y-Axis</summary>
    <h3>X-Axis</h3>
    <p>The x-axis is like a horizontal line on a graph, showing categories or time.</p>
    <p>Field: Choose the data column to display on the X-axis.</p>
    <p>Orientation(Bottom/Top): Decide where the X-axis labels and ticks appear</p>
    <p>Show Axis Button: Toggle to display or hide the X-axis.</p>
    <p>Stroke: Stroke refers to the thickness of the X-axis line</p>
    <p>Stroke Color: Choose the color of the X-axis line.</p>
    <p>Axis Type: Specify if the X-axis represents categories or numerical values.</p>
    <p>Angle: Adjust the angle of the X-axis labels for better readability.</p>
    <p>Height: Set the height of the X-axis line for visual consistency.</p>
    <h3>Y-Axis</h3>
    <p>The y-axis is a vertical line showing values or counts related to those categories or time.</p>
    <p>Field: Select the data column to display on the Y-axis.</p>
    <p>Aggregation: Choose how data on the Y-axis should be aggregated (e.g., sum, average).</p>
    <p>Show Axis Button: Toggle to display or hide the Y-axis.</p>
    <p>Axis Orientation: Specify whether the Y-axis labels and ticks appear on the left or right side.</p>
    <p>Axis Color: Set the color of the Y-axis line.</p>
    <p>Bar Color: Choose the color of bars in a bar chart on the Y-axis.</p>
    <p>Stacked: Top of each other bar / area for compactness. (Applicable to Bar , Area Chart)</p>
    <p>Stacked Button: Determine if bars / areas in the chart should be stacked.</p>
    <p>Show Bar / Area / Line Button: Toggle to display or hide bars / areas in the visual.</p>
    <p>Axis Type: Define if the Y-axis represents categories or numerical values.</p>
    <p>Area / Line Type: Select the Line / Area type design (Basis , Bump , Natural , etc.) (Applicable to Area , Line Chart)</p>
    <p>Add Y-axis: Allows you to add a secondary Y-axis to your visual, useful for comparing different data series with different scales on the same chart. (Can add up to 4 Axis)</p>
  </details>

  <details>
    <summary>Customize Visual Style</summary>
    <p>Legend: Display information about the Y-Axis and secondary Y-Axis</p>
    <p>Show Legend: Toggle to display or hide the legend.</p>
    <p>Position: Choose where the legend appears relative to the visual.</p>
    <p>Alignment: Align the legend items horizontally or vertically.</p>
    <p>Tooltip: Provides additional information when hovering over data points.</p>
    <p>Show Tooltip: Toggle to display or hide tooltips.</p>
    <p>Style Config: Adjust margins around the visual.</p>
    <p>Top/Right/Left/Bottom Margin: Set the spacing between the visual and its container.</p>
    <p>Grid: Adds a grid for better data interpretation.</p>
    <p>Show Grid: Toggle to display or hide the grid lines.</p>
    <p>Show Horizontal Grid: Toggle to show or hide horizontal grid lines.</p>
    <p>Show Vertical Grid: Toggle to show or hide vertical grid lines.</p>
  </details>

    <p>Aggregation: Choose how to aggregate data (e.g., sum, average) for calculating percentages.</p>
    <p>Show Button: Toggle to display or hide the percentage values on the chart.</p>
    <p>Is Animation Active Button: Toggle to enable or disable animations for the chart.</p>

# Building Cards and Tables

Building card and table visualizations involves crafting concise data summaries and structured information displays, facilitating quick insights and informed decision-making. These visual elements enhance data comprehension and user engagement within Metric Visualizer. 

If you need help determining the right chart for your data, refer to this documentation for guidance 

:point_right: Choosing the Right Visualization for Your Data

<div class="partner-details">
  <h2>Infographic Cards</h2>
  <details>
    <summary>Configuring Your Visual</summary>
    <p>Dataset: Choose the dataset on which your visual will be based.</p>
    <p>If you've uploaded a new dataset, click the refresh button to ensure it's up-to-date and available for visualization.</p>
    <p>Card Type: Select the orientation for your card: vertical or horizontal.</p>
    <p>Title: Set a concise title that describes your visual's purpose.</p>
    <p>Sub Title: Set a short description that describes your visual's purpose.</p>
    <p>Icon: Customize the icon based on the data you wish to showcase on the card.</p>
    <p>Filters: Apply filters to narrow down the data for your insights. Combine multiple filters using 'and' or 'or' operators.</p>
    <p>Operation: Choose a conditional operator (e.g., ==, >=) for your filter.</p>
    <p>Field: Select the column to which you want to apply the filter.</p>
    <p>Value: Enter the filter value to refine your data selection.</p>
  </details>

  <details>
    <summary>Set up your Data</summary>
    <p>Field: Choose the data column to be displayed or aggregated in the infographic card.</p>
    <p>Aggregation: Select how to aggregate the data (e.g., sum, average) for display in the infographic card.</p>
  </details>

  <details>
    <summary>Customize Visual Style</summary>
    <p>Content Alignment: Adjusts the alignment of text and other content within the infographic card.</p>
    <p>Icon Alignment: Aligns icons (Number, Currency, etc.) within the infographic card for visual consistency.</p>
