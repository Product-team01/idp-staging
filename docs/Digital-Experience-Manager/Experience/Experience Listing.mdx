<p>Experience created can be visualized in the Listing Table in the Experience Screen. The listing enables you to manage, search, sort, filter, and delete Experiences.</p>

<h2>Getting to Experience Listing</h2>

<div className="image-container">
  <img src="https://d1r1e7xjkfj7nz.cloudfront.net/experiencelisting1b.png" alt="Experience Listing" title="Experience Listing" />
</div>

<h3>Navigation:</h3>
<ul>
  <li>From the Vue Home Page -> Click on Digital Experience Hub to visit the Digital Experience Home Screen.</li>
  <li>Click on Experience on the navigation bar to access the Experience listing table.</li>
  <li>Alternatively, hover over the Digital Experience Hub on the navigation bar and click on Experience.</li>
</ul>

<h3>Search Experience</h3>
<ol>
  <li>You can search for the created Experience using the Experience name via the search bar or use one of the suggested/recently searched keywords.</li>
  <li>You can find the search results populated in the table.</li>
</ol>

<h3>Sort Columns</h3>
<ol>
  <li>Hover over the column header to find the sort icon next to each column.</li>
  <li>Click the Sort icon to sort the column alphabetically either in ascending or descending order.</li>
  <li>If the sort is applied to multiple columns, the column for which the sort was applied last will take precedence.</li>
</ol>

<h3>Filter Columns</h3>
<ol>
  <li>Hover over the column header to find the filter icon next to each column.</li>
  <li>Click the filter icon to open up a dropdown from which you can multi-select the values to be filtered.</li>
  <li>You can also search for a filter value within the dropdown.</li>
  <li>The table will be populated with the filtered results.</li>
</ol>

<h3>Delete Experience</h3>
<ol>
  <li>From the Experience listing table, next to each Experience name, click on the Delete icon under actions.</li>
  <li>Clicking Delete will prompt the user with an overlay modal that lists all the entities (Pages) the Experience is linked with.</li>
  <li>Here, you can access any entity config screen, make necessary changes before deleting the Experience.</li>
  <li>Deleting the Experience will unlink it from all the linked features and permanently delete it & the content from our system.</li>
</ol>

<h3>Publish / UnPublish</h3>
<ol>
  <li>From the Experience listing table, next to each Experience name, click on the Publish icon under actions to publish and unpublish an Experience.</li>
</ol>
