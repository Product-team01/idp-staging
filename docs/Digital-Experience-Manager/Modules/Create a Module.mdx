<p>Module enables you to (i) combine a strategy/combination of strategies/contents with a template, (ii) configure the number of results to be shown, (iii) link strategy(s)/content(s) to specific times on the template.</p>

<p><strong>Navigation</strong></p>
<ol>
  <li>To access modules navigate to the Vue menu bar, click on this icon</li>
  <li>Select Digital Experience Manager</li>
  <li>Select Module</li>
</ol>

<h2>Module Listing Screen</h2>

<p>The Module Listing screen provides you with the list of modules created in your account. From the listing screen, you can request creation of a new module, view details about each module configuration, Preview and Delete a module.</p>

<h2>Viewing a Module config</h2>
<ol>
  <li>From the Module listing table, next to each Module name, click on the 'Info' icon under actions.</li>
  <li>Module configuration details will open in an overlay modal.</li>
  <li>From here, you can also access all the entities linked (Strategy, Template, Experience) to this Module and navigate to them.</li>
</ol>

<h3>Search Modules</h3>
<ol>
  <li>You can search for the created Modules using the Module name via the search bar or use one of the suggested/recently searched keywords.</li>
  <li>You can find the search results populated in the table.</li>
</ol>

<h3>Sort Columns</h3>
<ol>
  <li>Hover over the column header to find the sort icon next to each column.</li>
  <li>Click the Sort icon to sort the column alphabetically either in ascending or descending order.</li>
  <li>If the sort is applied to multiple columns, the column for which the sort was applied last will take precedence.</li>
</ol>

<h3>Filter Columns</h3>
<ol>
  <li>Hover over the column header to find the filter icon next to each column.</li>
  <li>Click the filter icon to open up a dropdown from which you can multi-select the values to be filtered.</li>
  <li>You can also search for a filter value within the dropdown.</li>
  <li>The table will be populated with the filtered results.</li>
</ol>

<h3>Delete Module</h3>
<ol>
  <li>From the Module listing table, next to each module name, click on the 'Delete' icon under actions.</li>
  <li>Clicking Delete will prompt the user with an overlay modal which lists all the entities (Experiences) this Module is linked with.</li>
  <li>From here, you can access any entity config screen, make necessary changes before deleting the module.</li>
  <li>Deleting the module will unlink it from all the linked features and permanently delete it & the content from our system.</li>
</ol>

<h2>Request creation of a New Module</h2>
<ol>
  <li>Navigate to Digital Experience Manager (DXM) via Vue Menu Bar and click on 'Assets' > Module</li>
  <li>Click on 'Request New'</li>
  <li>In the form provided please fill out the necessary details. And we will get back to you with your Module within 7 to 14 business days.</li>
  <li>Details to be provided:</li>
</ol>

<ul>
  <li>Module Type:
    <ul>
      <li>Embed Code</li>
      <li>API</li>
      <li>Email</li>
    </ul>
  </li>
  <li>Template</li>
  <li>Strategy(s)</li>
  <li>Min & Max number of items</li>
</ul>
