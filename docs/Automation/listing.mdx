## Search

![Search](https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-105430.png)
Search across Workflow names and statuses using the search bar. Alternatively, use suggested/recently searched keywords to get to what you are looking for faster.

## Sort Columns

![Sort Columns](https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-105402.png)
Hover on the column header to find the sort option to the right of a column header.

Click the Sort icon to sort the column alphabetically in either ascending or descending order.

If the sort is applied to multiple columns, the column for which the sort was applied last will take precedence.

## Filter Columns

![Filter Columns](https://d1r1e7xjkfj7nz.cloudfront.net/image-20240520-105516.png)
Hover on the column header to find the filter option to the left of a column header.

Click the filter icon to open up a dropdown from which you can single select or multi-select the values to be filtered.

You can also search for a filter value within the dropdown.

## Delete Workflow

From the Workflow listing table, next to each Workflow name, click on the Delete icon under actions.

Clicking Delete will prompt the user with an overlay modal to confirm this action.

Deleting the Workflow will permanently remove it and the content from the system.
